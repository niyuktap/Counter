
🔢 Counter Webpage
A simple and interactive counter webpage built using HTML, CSS, and JavaScript. Users can increment, decrement, and reset the count in real time. It demonstrates DOM manipulation, event handling, and responsive design, making it a great beginner-friendly project.

---

🚀 Features
- ➕ Increment counter  
- ➖ Decrement counter  
- 🔄 Reset functionality  
- ⚡ Real-time updates  
- 📱 Responsive design  

---

🛠️ Tech Stack
- HTML  
- CSS  
- JavaScript  

---

📂 Project Setup
1. Clone the repository:
   git clone https://github.com/niyuktap/Counter.git
2. Open the project folder
3. Run `index.html` in your browser
